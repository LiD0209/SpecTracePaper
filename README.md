# SpecTracePaper

USENIX Security 2026 LaTeX paper source.

## Build

From the repository root:

```powershell
cd usenixsecurity2026_latex_templates
latexmk -pdf usenixsecurity2026.tex
```

The main paper source is `usenixsecurity2026_latex_templates/usenixsecurity2026.tex`, with section files under `sections/`.
